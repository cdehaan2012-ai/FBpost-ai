# 🕊️ FamPost AI - Christian Visual Generator

**FamPost AI** adalah aplikasi web modern berbasis kecerdasan buatan (AI) yang dirancang untuk membantu umat Kristiani membuat konten visual estetik dengan cepat. Aplikasi ini menggabungkan ayat-ayat Alkitab dan kutipan rohani dengan latar belakang gambar yang relevan secara otomatis.

---

## ✨ Fitur Utama

- **Kategori Konten Spiritual**: Pilih dari berbagai kategori seperti Ayat Alkitab, Renungan Harian, Janji Tuhan, dan Motivasi Kristen.
- **Mood & Tone Adaptif**: Sesuaikan nuansa konten dengan pilihan mood Menyejukkan (Damai), Memberkati (Kasih), Menguatkan (Iman), atau Sukacita (Pujian).
- **Generator Gambar Cerdas**: Sistem pencarian gambar otomatis berbasis kata kunci yang sinkron dengan kategori dan mood yang dipilih.
- **Kustomisasi Layout**: Dukungan berbagai rasio gambar (1:1 untuk Post, 9:16 untuk Story), serta berbagai pilihan font premium.
- **Branding Personal**: Tambahkan watermark atau nama akun media sosial Anda sendiri ke dalam desain.
- **Riwayat Desain**: Simpan hasil karya Anda di penyimpanan lokal browser untuk diakses kembali kapan saja.
- **Export Berkualitas Tinggi**: Unduh hasil desain dalam format PNG kualitas tinggi yang siap dibagikan ke media sosial.

---

## 🚀 Teknologi yang Digunakan

- **Core**: Vanilla JavaScript (ES6+), HTML5, CSS3.
- **Design System**: Glassmorphism UI dengan CSS kustom dan Google Fonts.
- **Image API**: Powered by [Unsplash](https://unsplash.com/).
- **Canvas Processing**: [html2canvas](https://html2canvas.hertzen.com/) untuk konversi DOM ke gambar.
- **Deployment**: [Vercel](https://vercel.com/) (CI/CD via GitHub).
